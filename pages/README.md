# Cardthropic Pages

This folder is a static site bundle for Codeberg Pages.

## Quick Publish (Codeberg UI)

1. Open repo `Settings` -> `Pages`.
2. Set source branch to `0.9.0pre` (or `main` later).
3. Set source directory to `/pages`.
4. Save.
5. Wait for publish and open your Pages URL.

## Notes

- Entry file: `pages/index.html`
- Stylesheet: `pages/style.css`
- The page references logo via absolute Codeberg raw URL so no extra asset copy is needed.
