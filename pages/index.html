At a Glance

Rust GTK4 Libadwaita GNOME GPL-3.0-or-later Zero permissions Flatpak

Cardthropic is a modern GNOME solitaire app focused on deterministic seeds, strong automation, and transparent private metrics for tuning and benchmarking.

Game Modes

Mode Status Notes
Klondike Playable Deal 1-5, seeds, Smart Move, Wand, Robot, W?
Spider Playable 1/2/3/4 suits, Smart Move, Wand, Robot, W?
FreeCell Playable Full gameplay loop with Smart Move, Wand, and Robot tooling

What It Includes

  • Zero permissions Flatpak by default
  • Klondike and Spider (1/2/3/4 suit modes)
  • Numeric and word seeds with deterministic mapping
  • Wand, Robot, Forever Mode, and automation strategy profiles
  • Benchmark snapshot private metrics and live status history
  • Undo/redo, session resume, custom CSS userstyles

Automation and AI

  • Automation strategy selector: fast, balanced, deep
  • Middle-click Robot starts Forever Mode continuous runs
  • Benchmark snapshots include wins, losses, win rate, memory, and strategy
  • Status history preserves full machine-readable run output

Install (Flatpak)

Install from the latest release artifact:

flatpak install ./cardthropic.flatpak
flatpak run io.codeberg.emviolet.cardthropic

Trust and Privacy

  • Zero permissions Flatpak runtime profile
  • No account required, no cloud sync required
  • Private metrics are local app signals for gameplay and benchmarking
  • Open source code and changelog are public and auditable
